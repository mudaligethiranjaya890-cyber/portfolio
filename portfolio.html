05 — Projects

Things I've built

Project 01

Battle Buddy 2.0: A Secure IoT-Based Real-Time Soldier Health and Localization Monitoring System with Predictive Intelligence

August 2025 – Present

Battle Buddy 2.0 is an IoT-based real-time soldier monitoring system designed to improve soldier safety during field operations. The system integrates health monitoring, location tracking, fall detection, secure communication, real-time dashboard visualization, and predictive intelligence.

Wearable sensor nodes collect soldier health and movement data including heart rate, SpO₂, body temperature, GPS location, and motion-based fall detection. Data is transmitted through secure LoRa communication to a gateway and displayed on a real-time dashboard.

Main Features
Real-time health monitoring GPS-based location tracking Kalman-filtered GPS localization Fall detection Secure LoRa communication Relay node support Real-time dashboard Predictive intelligence
My Contribution

Focused on secure LoRa communication, networking, data transmission, and system integration. Worked on reliable long-range communication, secure packet handling, relay-based communication, and data transfer from field node to dashboard infrastructure.

Technologies Used
ESP32 LoRa SX1278 GPS Module MPU6050 MAX30102 MLX90614 Node-RED React.js Cybersecurity Predictive Models
